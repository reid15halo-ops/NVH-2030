Overview

What It Does

MiR250 — Autonomous Mobile Robot

  • AMR (Autonomous Mobile Robot): uses LiDAR, 3D cameras, and SLAM to navigate freely — no fixed paths or magnetic tape
  • Payload: 250 kg — sufficient for MJF build units (~350 kg with top module) or part bins
  • Self-navigating: dynamically avoids obstacles, adapts to layout changes, integrates with doors/elevators
  • Fleet management software for multi-robot coordination — can serve multiple workcells
  • MiR is a Teradyne company — global support, automotive-grade reliability
  • AMR market growing at 32.7% CAGR — mainstream technology in modern factories by 2026
How It Works

Process & Principle

Automated Material Flow

  • Route 1: Printer → Cooling station (build unit transport after print completes)
  • Route 2: Cooling station → Processing Station (depowdering after cooldown)
  • Route 3: Post-processing → QC/Packing (finished parts to clean zone)
  • Route 4: Packing → Shipping dock (labeled HUs to dispatch)
  • Trigger: Apriso MES sends mission to MiR Fleet when job status changes
  • Integration: REST API connects MiR Fleet to Apriso — fully automated mission dispatch

Gundernhausen Fit

Relevance & Recommendation

Recommendation for Gundernhausen

  • Phase 1–2: NOT needed. With <30 employees and one workcell, manual transport is sufficient
  • Phase 3–4 (Month 13+): Evaluate when 2+ printers running and transport becomes a bottleneck
  • Pilot: Start with 1× MiR250 (€58–83k deployed) for build unit shuttle between zones
  • ROI: Marginal in small operation — primary value is 24/7 autonomous operation for lights-sparse concept
  • Alternative: Simple roller conveyor between printer and processing station (€5–10k) may suffice
