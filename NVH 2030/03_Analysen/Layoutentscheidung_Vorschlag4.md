**Status:** AUSGEWÄHLT
**Datum:** 26.02.2026
**Standort:** Werk Gundernhausen, Deutschland

---

## Entscheidung

**Vorschlag 4 wurde als Basislayout für die Umsetzung ausgewählt.**

### Begründung
1. Einziger Vorschlag mit expliziter CO2- und Trennmittel-Zonenplanung
2. Drei Produktbereiche klar getrennt durch Staplerverkehrswege
3. Höchste Flexibilität für zukünftige Technologieintegration (Bio-Polyol, PUSH HR, CO2-RA)
4. Beste Bewertung in allen 5 Kriterien (Gewichtete Gesamtnote: 4,0 von 5,0)
5. Marginaler Kostenunterschied gegenüber Vorschlag 2 (<4%)

## Detaillierte Zonenanalyse — Vorschlag 4

### Zonenübersicht

Vorschlag 4 teilt Halle 2 in **drei Produktbereiche**, getrennt durch **Staplerverkehrswege** (schwarz umrandet):

│                                                                  │
│  ┌─────────────────────────────┐    ┌──────────────────────────┐ │
│  │  PRODUKTBEREICH 1 (OBEN)    │    │  Büros, Elektro-Werkstatt│ │
│  │  Gemischte Produktionszellen│    │  Technische Betriebe     │ │
│  │  • Produktgruppe A          │    │  Aufenthaltsraum         │ │
│  └─────────────────────────────┘    └──────────────────────────┘ │
│  ═══════ STAPLERVERKEHRSWEG ═════════════════════════════════    │
│  ┌─────────────────────────────────────────────────────────────┐ │
│  │         PRODUKTBEREICH 2 (MITTE) — HAUPTPRODUKTION          │ │
│  │                                                              │ │
│  │  • Gemischte Produktionszellen (Boxen, Tandem, etc.)        │ │
│  │  • Dosier- & Mischanlagen                                    │ │
│  │  • [CO2] — Aktuell: Treibmittel / Zukunft: RA-Versorgung   │ │
│  │  • [Trennmittel] — Aktuell: wässrig / Zukunft: CO2-basiert │ │

**Wichtig:** Die schwarz umrandeten Bereiche definieren die **Produktbereiche**, getrennt durch **Staplerverkehrswege**. Dies ermöglicht sicheren Gabelstaplerverkehr zwischen den Bereichen ohne Kreuzung mit Fußgängerwegen.

## Staplerverkehrskonzept

Die schwarz umrandeten Zonen in Vorschlag 4 definieren **sichere Staplerverkehrswege** zwischen den drei Produktbereichen:

```
     ROHSTOFF-ANLIEFERUNG
            │
            ▼
  ┌── PRODUKTBEREICH 1 ──┐
  │   (Produktgruppe A)   │
  └──────────┬────────────┘
     ════ STAPLERWEG ════
  ┌── PRODUKTBEREICH 2 ──┐
  │   (Hauptproduktion)   │
  │   CO2 | Trennmittel   │
  └──────────┬────────────┘
     ════ STAPLERWEG ════
  ┌── PRODUKTBEREICH 3 ──┐
  │   (Produktgruppe C)   │
  └──────────┬────────────┘
            │
            ▼
       VERSAND / LAGER
```

**Vorteile:**
- Staplerverkehr klar von Fußgänger-/Bedienbereichen getrennt
- Drei Produktbereiche unabhängig anfahrbar
- Rohstoffe und Fertigteile über definierte Wege transportiert
- Sicherheit: Keine Kreuzungen Stapler/Fußgänger innerhalb der Produktbereiche
